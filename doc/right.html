Report manager is a reporting application like Crystal Reports (Report Manager Designer) and also a set of components for Delphi/Kylix

It's opensource under the MPL license model so you can use it in your commercial software but any enhacements to the engine must be published.

Current version is 0.97 beta (currently supports Delphi6 & Kylix 2)

Near all features are implemented, the few one for version 1.0 are:

  • Delphi 5/C++Builder Support (removing the qtintf.dll dependence in Windows).
  • Maybe barcode printing
  • Maybe some charting
