Report Manager Reporting Tool

Version 1.1

Supports Delphi 5-7,Kylix 2-3,Builder 4-6

You need a designer to create reports.

You can find reportmanager designer at:

http://sourceforge.net/projects/reportman


More info and installation instructions (tutorial):

http://reportman.sourceforge.net

- TCP Report Server and Thin Report client.
- Windows NT Report server service and linux server.
- 100% Windows XP support.
- No Qt dependences in non CLX apps (Windows)
- Images loaded from fields pointing to files.
- Few general enhacements.
- Few bugfixes.
