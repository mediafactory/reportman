Using report parameters

Report parameters is a powerfull feature, here is a list of samples you can do with parameters

  • Allow the user to change words or sentences of the report
  • Set initial values of group sums...
  • Assign them to querys to alter the result set (parametrized querys)
  • The user can alter the parameters that are marked as visible
  • Print them, or alter them while executing the report
