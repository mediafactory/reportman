Report Manager Web Server

Operation

The web server has predefined commands

The entry point por the web server is the login page:

http://localhost/cgi-bin/repwebserver.dll/login

To test installation you can call version command

http://localhost/cgi-bin/repwebserver.dll/version

To obtain additional information about the web server you can use the info command.

http://localhost/cgi-bin/repwebserver.dll/info

After login you can execute configured reports or configure the web report server (if you logged in as Admin)
