Introduction

Missing features, maybe included in next major versions

This is a missing features list, it can be considered also a to do list.

General
  • Exporting to HTML format (for now you can export to pdf, then use a pdf2html utility)
  • Passing parameters and other configuration options to command line tools using a file.
  • Exporting to TXT format, you can use now a generic text printer in windows and a ps2txt filter in print queue in Linux.
Supported database engines
  • Native Zeos Database support (is supported by passing the datasets to the engine)
  • Removing the TClientDataset dependence (it will allow compile with Professional versions of Delphi 4/5 and Builder 4/5
Report Server
  • Better administration of connections, list connections, and the state of each connection, kill connections...
Report definition & desingning
  • Better report structure representation
Printing engine
  • Print a large text across pages
  • Print formated text (RTF)
Properties for all controls
  • Positioning based on expressions
Properties for text controls
  • Print across pages in large text controls
Properties for expressions
  • More functions available
Image print control
  • Monocrome and RLE compressed bitmaps in PDF export (Qt and GDI already support them)
Barcode control
  • Enhaced barcode printing in Linux with a better resolution (Qt 3.0 version is needed to pass the 100dpi limitation)
Chart control
  • More chart types, for now only lines is implemented, may be include TeeChart support
Simple drawing/decoration control
  • Polylines
