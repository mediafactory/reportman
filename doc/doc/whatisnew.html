Release notes

What is new

Version 1.4 include this new features

  • Positioning of sections based on expressions
  • Page positioning of sections based on expressions (two pass report required)
  • The features above allow printing of datasets in paral.lel.
  • Bugfix to support two pass report in web/tcp server
  • French language support, thanks to Jean Louis Argente (argente@alussinan.org)
  • Report libraries, creating, altering, storing reports inside databases
  • Report wizard

  • Web report server
  • TeeChart Support
  • External sections (database streamable)
  • Child-cascade subreports
  • Enhaced Adobe PDF image support
  • Enhaced documentation

Other minor feature enhacements:

  • Enhaced expression build wizard
  • Small Bugfixes
  • OnBeforePrint event handling (Delphi/Kylix/Builder)
  • Enhaced main websit
