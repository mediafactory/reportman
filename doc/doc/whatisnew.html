Release notes

What is new

Version 2.1 include this new features

  • Background image in sections for easy form filling and printing.
  • E-mail report (pdf) feature from preview
  • Enhaced Teechart properties
  • Bug fixes

Version 2.0 include this new features

  • Native .Net support, compiled with Delphi for .Net.
  • HTML output
  • Full Report Library support (reports stored inside database tables with a tree structure)
  • Enhaced security model for web server and TCP report server
  • Enhaced ActiveX
  • Enhaced designer,data browser with drag&drop feature, categorized object inspector, documentation and more.
  • Report compiler (Windows only), generate a executable independent report (About 300Kbytes).
  • Enhaced Borland C++ Builder 6 support.
  • Some of minor features not listed here.
  • Notes: This upgrade requires updates for client and server, that is a 1.6 client does not work with a 2.0 server and a 2.0 client does not work with 1.6 server.

Version 1.6 include this new features

  • Multiprocessor support (smp).
  • New output driver, text only with printer escape codes to print lighting fast in any dot matrix printer, in Windows and Linux, also to save the report result as plain text.
  • New shared libraries libreportmanapi.so and libreportmanapiqt.so to access the engine with GNU C and any language capable of calling dynamic link library functions (Reportman.ocx for Windows)
  • Zeos library drivers support
  • Excel export support for Microsoft Windows
  • Allow report definition streaming in three formats binary-zlib, binary and text , so you can track changes in your reports with cvs
  • Plain text file input support as a database file with visual configuration dialog
  • New method for VCL components and ActiveX, executeremote to execute reports stored in report servers, also available in ActiveX and GNU C interface.
  • New property editors for Delphi 5 (TRpAlias)
  • Enhaced command line tools (-preview -pdialog options added), all tools can work now with standard input and standard ouput.
  • ActiveX composite reports feature.
  • New methods for drawing text and graphic elements usable from the expression evaluator.
  • Enhaced designer - New database connections/datasets configuration dialog, active help in properties in the object inspector...
  • Enhaced support for Borland C++ Builder (added CLX package and more documentation)
  • Enhaced documentation
  • Portuguesse language support thanks to Warlei Alves
  • Bug fixes
  • Report Manager is now a Project Jedi Alliance member
  • Notes: This upgrade requires updates for client and server, that is a 1.5 client does not work with a 1.6 server and a 1.6 client does not work with 1.5 server. This is also true for ActiveX Internet explorer plugin. Replace dbxdrivers old file with the new one for zeos support.

Version 1.5 include this new features

  • Client side unions with grouping feature to support cross database dataset unions
  • Command line tools support now the parameter -paramPARAMNAME=paramvalue
  • Parameter values from alist of provided values
  • Full Bidi support for MS Windows, partial Bidi support for PDF and Linux.
  • Global scope page headers and footers
  • Printing large text across multiple pages is now implemented (Multi Page property)
  • Barcode square rotation
  • ActiveX interface to report parameters and parameter information
  • Enhaced Designer

Version 1.4 include this new features

  • Positioning of sections based on expressions
  • Page positioning of sections based on expressions (two pass report required)
  • The features above allow printing of datasets in paral.lel.
  • Bugfix to support two pass report in web/tcp server
  • French language support, thanks to Jean Louis Argente (argente@alussinan.org)
  • Report libraries, creating, altering, storing reports inside databases
  • Report wizard

Version 1.3 include this new features:

  • Web report server
  • TeeChart Support
  • External sections (database streamable)
  • Child-cascade subreports
  • Enhaced Adobe PDF image support
  • Enhaced documentation
  • Enhaced expression build wizard
  • Small Bugfixes
  • OnBeforePrint event handling (Delphi/Kylix/Builder)
  • Enhaced main website
  • Added source code units documentation
