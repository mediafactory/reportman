Installation

Compiling Report Manager using C++Builder

Download the components from sourceforge and unpack it to a directory:

http://sourceforge.net/projects/reportman

Version Packages Description
Builder 4 rppack_builder4.dpk Unique complete package (no report design)
Builder 6 rppack_builder6.dpk Base package, non-visual, platform independent
rppackv_builder6.dpk Visual CLX package, for cross platform development
rppackvcl_builder6.dpk VCL package can be used only in Windows
rppackdesign_builder6.dpk Visual CLX interface to Report Manager Designer

You must install the packages in the correct order.

You must add the report manager components directory to the library path.
