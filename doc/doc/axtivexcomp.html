Installation

Using Report Manager ActiveX control

Download the activex component from sourceforge and unpack it to a directory:

http://sourceforge.net/projects/reportman

Unzip it and install it (you must have Administrator Rights)

At command line execute:

regsvr32 TAXReportXControl1.ocx

In your development enviroment you usually must activate it to use, is called TAXReport Control or TAXReportXControl1 Library.

Then the control is available to insert in any ActiveX application or development enviroment, usually you use Filename, Preview property and Visible properties at designtime and Execute method at runtime.
