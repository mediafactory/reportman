Installation

Report Manager Server

Installation in Microsoft Windows as a Service

If you are running Windows NT/2000/XP you can install the server as a standalone service, you must run repserverservice /INSTALL, you shoud have a shortcut available in the start menu inside Report Manager group. To unistall it you can run repserverservice /UNINSTALL or the shortcut menu.

After installing the service you can set startup options in the same way as other services.

Installation in Microsoft Windows as a standalone application

If you are running Windows 98/Me you can add a shortcut to reportserverapp.exe in the Start group so it's started after logging. You can also execute reportservercon.exe, to start it as a console application.

Installation in Linux as a standalone application

To start the server you must execute reportserverapp.sh or reporservercon.sh script, the first need an X Server running because is a X application.

Installation in Linux as a service

You can run the report server at boot time, the procedure depends in you linux distribution, usually the runlevel to start the server is 3 (console network), that is execute reportservercon.sh.
