Report Manager is a reporting scheme designer and a printing engine.

  • Opensource under the MPL license model
  • Cross platform supporting Windows and Linux (Kylix/Delphi)
  • Database access
    • Windows: Borland database engine (DBase, Paradox, Interbase, Oracle, DB2, SQL Server, ODBC)
    • Windows: Microsoft Data Access Objects (Microsoft Jet, ODBC...)
    • Windows & Linux: DBExpress (Interbase, MySQl, PostgreSQL, DB2, Oracle, ODBC)
    • Windows & Linux: Native Access to Interbase (IBX, no additional libraries needed)
    • Windows & Linux: Borland Native simple local file database (MyBase)
  • Report definition
    • Unlimited number of subreports, each subreport can use a dataset
    • Subreports can be used as report headers or summary information
    • Each subreport can have a unlimited number of sections
      • Page headers
      • Page footers
      • Grouping (unlimited), a group changes when a expression is true or changes
      • Details. All details are printed once for each record of the subreport's dataset
    • Each section has also some properties
      • Print condition
      • Skip page before or after print
  • Printing engine
    • Horizontal movement (labels)
    • For text controls
      • Font Name (for each platform)
      • Font Size, Bold, Italic, Color.
      • Background color or transparent
      • Alignment, multiline
      • Display Mask
    • Page Margins and page orientation
    • Image print support
      • The image is embedded in the report
      • The image can be loaded from a binary field
      • The image can be stretched, or adapt to pixels per inch
    • Simple graphic support
      • Horz line, Vert line, Rectangle, Ellipse, Roundrect...
      • Pen style (dash,dash dot, solid) and color
      • Brush Style (lines, cross, solid, pattern) and color
    • Expression evaluator, you can use fields and functions
    • Preview
      • Fast preview
      • Save the report as a metafile report (device independent graphic format)
      • A separate application can print/preview metafiles (metaview)
