Servidor de informes web

El servidor web Report Manager genera archivos pdf al vuelo, preguntando al usuario los parámetros del informe, ejecuta el informe y devuelve un archivo con formato Adobe PDF, si su navegador dispone del plugin adecuado podrá visualizar el archivo dentro de la ventana de su navegador.

Demostración de servidor web

Haga click aquí para ver un servidor de informes web en acción (Entre Usuario Admin y la contraseña en blanco).

Nota: El servidor muestra los mensajes en inglés en esta demostración.

La máquina que ejecuta la aplicación:

Procesador Intel Pentium
Velocidad de reloj 133 Mhz
RAM 48 Megabytes
Sistema Operativo Suse Linux 7.3
Servidor Web Apache
Conexión a Internet ADSL (256Kbits/128Kbits)
